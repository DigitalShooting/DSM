Verwaltungsoberfläche für DSCs, basierend auf der Socket.io API.

## Funktionen

### Stammdaten
Verwaltung der Schützen

### Stände
Verwaltung der Stände:
- Ein/ Ausschalten
- Disziplin
- Part
- Schütze
- Drucken
